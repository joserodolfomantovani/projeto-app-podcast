# projeto-app-podcast

## Descrição

Este projeto é um aplicativo de podcast desenvolvido em Node.js sem frameworks. Ele permite que usuários descubram, ouçam e gerenciem seus podcasts favoritos de forma simples e intuitiva.

## Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)  
- [TypeScript](https://www.typescriptlang.org/)  
- [TSX](https://github.com/esbuild/tsx)  
- [Tsup](https://tsup.egoist.dev/)  
- [@types/node](https://www.npmjs.com/package/@types/node)
